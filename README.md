## Installation

```bash
# git clone, then remove the .git
git clone https://github.com/leedom92/vue-h5-template.git

# or you can use degit, .git is removed automatically
npx degit leedom92/vue-h5-template vue-h5-template

cd vue-h5-template

pnpm install

pnpm serve
```
